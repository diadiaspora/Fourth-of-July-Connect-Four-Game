# Connect Four Game 🎮

A browser-based implementation of the classic Connect Four game built using HTML, CSS, and JavaScript. This project demonstrates DOM manipulation, game logic, responsive design, and creative UI styling.

---

## 🌐 Live Demo

Play the game here:  
https://diadiaspora.github.io/Fourth-of-July-Connect-Four-Game/?

---



---

## 🚀 Features

- Two-player turn-based gameplay (Red vs Yellow)
- Win detection (horizontal, vertical, and diagonal)
- Draw detection when the board is full
- Dynamic DOM updates with JavaScript
- Responsive layout for different screen sizes
- Interactive hover effects for better user experience

---

## 🎨 Custom Styling

This version includes a themed visual redesign inspired by **Fourth of July celebrations**, featuring:

- Patriotic red, white, and blue color accents
- Firework-inspired CSS animations
- Confetti effects for enhanced game feedback
- Improved board styling for a modern look

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox / Grid / Animations)
- JavaScript (Vanilla DOM manipulation)
